# Alumni Connect

A web-based alumni networking and engagement platform built with Next.js, TypeScript, and Supabase.

## Features

- Alumni networking and community building
- Structured mentorship programs
- Job board and opportunity sharing
- Fundraising and donation system
- Admin analytics dashboard

## Tech Stack

- **Frontend**: Next.js 14 with TypeScript
- **Styling**: Tailwind CSS
- **Backend**: Supabase (PostgreSQL + Auth + Storage)
- **Deployment**: Vercel (recommended)

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up Supabase:
   - Create a new Supabase project
   - Copy your Supabase URL and anon key to `.env.local`
4. Run the development server:
   ```bash
   npm run dev
   ```
